This applet is run by loading BowlingApplet.html.
Add the url parameter ?code=true to view the code block that shows the requirements to get 100% code coverage.

The two javascript files are:
    BowlingApp.js which handles the applet and its interactions.

    BowlingGame.js defines the BowlingGame object that is used throughout the BowlingApp.js code. 
    It defines what is needed to calculate code coverage.
